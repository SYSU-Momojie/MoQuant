# MoQuant

## Dependencies
package|version
|:----:|:----:|
|python|3.7|
|pip|19.2.3|
|pymysql|0.9.3|
|pandas|0.25.1|
|tushare|1.2.45|
|SQLAlchemy|1.3.8|
|lxml|4.4.1|
|requests|2.22.0|
|bs4|0.0.1|


## Database - MySql
